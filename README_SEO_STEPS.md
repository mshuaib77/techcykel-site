# TechCykel SEO Pack

Upload these files to the root of the `techcykel-site` GitHub repo, replacing the current `index.html` and keeping `success.html`.

Files included:
- index.html
- success.html
- robots.txt
- sitemap.xml
- _redirects
- mobile-data-destruction-nyc.html
- e-waste-pickup-nyc.html
- it-asset-disposal-nyc.html
- secure-document-destruction-nyc.html

After Netlify deploys:
1. Open https://techcykel.com/robots.txt
2. Open https://techcykel.com/sitemap.xml
3. Add techcykel.com to Google Search Console as a Domain property.
4. Submit sitemap: https://techcykel.com/sitemap.xml
5. Use URL Inspection and request indexing for the homepage and each service page.
